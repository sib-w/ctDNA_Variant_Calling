ctDNA_Variant_Calling

Description:
This repository contains the results of a ctDNA variant calling pipeline. .ipynb files, VCF variant files and QC/trimmed HTML reports are included.

Folder Structure:
- .ipynb           : .ipynb files
- *.vcf            : Variant Call Format files
- qc/              : FastQC HTML reports
- trimmed_html/    : fastp HTML reports
- README.txt       : Project documentation

Usage:
1. Clone the repository:
   git clone https://github.com/sib-w/ctDNA_Variant_Calling.git
   cd ctDNA_Variant_Calling

2. Open HTML reports in a browser to inspect QC and trimming results.
